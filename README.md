# 新天地棋牌组队系统

这是一个 macOS 本地第一版应用，用来管理麻将客户库、每日报名名单和四人组队。

## 使用方式

双击 `新天地棋牌组队系统.app` 启动。

当前版本是原生 macOS 应用壳，不再使用旧的 Python/Tk 启动窗口。

## 当前功能

- 客户总库：手动新增、编辑、删除客户。
- 自定义属性：可以按门店需要添加属性，例如牌技、常来时段、消费习惯。
- 冲突关系：设置谁和谁不能同桌，系统会双向记录。
- 每日报名：按日期从客户总库挑选当天报名名单。
- 自动组队：只从当天报名名单中选择，每四人为一桌，并避开冲突客户。
- 本地保存：数据保存在应用本地存储中。

## 后续可升级方向

- 导入/导出 Excel。
- 会员充值、消费记录、到店频率统计。
- 更细的组队规则，例如牌技均衡、男女比例、熟客带新客。
- 打包成正式 `.app` 应用图标。
